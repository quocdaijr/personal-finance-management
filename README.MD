# Comprehensive Personal Finance Management Project

This project is a comprehensive personal finance management application aimed at providing a wide range of financial tools, including transaction tracking, budgeting, forecasting, investment management, debt management, tax planning, and more.

## Technology Stack
- **Backend:** Golang (Gin), PostgreSQL, Redis
- **Analytics Service:** Python (FastAPI, Pandas, scikit-learn)
- **Frontend:** React (Material-UI)
- **DevOps:** Docker, Kubernetes, GitHub Actions CI/CD, Terraform
- **Others:** JWT, OAuth

## Repository Structure
```
├── backend/         # Golang microservices
├── analytics/       # Python data analytics service
├── frontend/        # React web application
├── infra/           # Infrastructure as code (Docker, Kubernetes, CI/CD)
├── docs/            # Project documentation
├── .gitignore
└── README.md
```

## Getting Started
1. Clone the repository.
2. Ensure you have Docker, Kubernetes (minikube/kind), and Go installed.
3. Follow the instructions in the `/infra` folder to set up the deployment environment.
4. Explore each directory for more details on setup and contribution guidelines.
